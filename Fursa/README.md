# Fursa - African Freelance Marketplace

**Fursa** is a modern job/talent marketplace platform designed for African freelancers and employers to connect, collaborate, and grow together. The platform bridges the gap between talented professionals across Africa and businesses seeking specialized skills.

## Features

- **Job Marketplace**: Browse and apply for freelance opportunities tailored to your skills
- **Talent Discovery**: Find skilled professionals across various disciplines
- **Smart Matching**: AI-powered job-to-profile matching based on skills and experience
- **Portfolio Showcase**: Display your work and attract quality opportunities
- **Real-time Communication**: Connect directly with employers and clients
- **Secure Payments**: Safe and reliable payment processing
- **Video Introductions**: Personal video intros to stand out from the crowd
- **Location-based Search**: Find opportunities and talent near you

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite with HMR for fast development

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Starts the development server at `http://localhost:5173/`

### Build

```bash
npm run build
```

Builds the app for production in the `dist/` folder.

### Preview

```bash
npm run preview
```

Previews the production build locally.
