# Simple Portfolio

A clean and minimal portfolio website to showcase your work, skills, and experience.

## Contents

- `index.html` - Portfolio page structure
- `style.css` - Custom styling and layout
- `README.md` - This file

## Features

- Clean, modern design
- Responsive layout
- Easy to customize
- Minimal dependencies

## Quick Start

1. Open `index.html` in your browser to view the portfolio
2. Edit `index.html` to add your content (name, bio, projects, links)
3. Customize `style.css` to match your personal brand and preferences

## Customization

- Update the HTML with your personal information, projects, and links
- Modify colors, fonts, and spacing in the CSS file
- Add images to the assets folder or use external URLs

## Browser Compatibility

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
